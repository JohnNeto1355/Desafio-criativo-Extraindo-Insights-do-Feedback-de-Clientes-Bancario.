# Desafio Criativo: Extraindo Insights do Feedback de Clientes Bancários

# Sobre o Desafio

Neste desafio, o objetivo foi criar um prompt capaz de ajudar uma Inteligência Artificial a analisar feedbacks de clientes bancários e transformar essas informações em insights úteis para a tomada de decisão.

A atividade demonstrou como a qualidade de um prompt depende de uma boa definição de objetivo, contexto e restrições.

# Definindo a Intenção

Primeiramente, defini o que eu queria que a IA fizesse e qual seria o resultado esperado.

Quero que a IA analise feedbacks de clientes bancários sobre serviços como aplicativo, Pix, cartão de crédito e atendimento digital para identificar dúvidas frequentes, elogios e oportunidades de melhoria.

O resultado será utilizado pela equipe de experiência do cliente para apoiar melhorias nos serviços e aumentar a satisfação dos usuários.

A entrega deverá conter um resumo executivo, uma tabela com os principais insights encontrados e recomendações de ações.

Considerarei o resultado satisfatório se ele for claro, organizado e baseado apenas nos dados fornecidos.

# Adicionando Contexto e Restrições

Para melhorar a qualidade da análise, adicionei informações sobre o contexto dos dados e os cuidados necessários.

# Contexto

Estou trabalhando com feedbacks de clientes relacionados a aplicativos bancários, Pix, cartão de crédito e canais digitais de atendimento.

# Dados disponíveis

A base de dados poderá conter:

Data do comentário;

Canal de atendimento;

Texto do feedback;

Produto citado;

Nota de satisfação de 1 a 5.

# Critérios de Análise

A IA deverá classificar os feedbacks por:

Tema;

Sentimento;

Urgência;

Produto citado;

Impacto na experiência do cliente.

# Restrições

Utilizar apenas os dados fornecidos;

Não inventar informações ou conclusões;

Não expor dados pessoais dos clientes;

Informar quando os dados não forem suficientes para uma análise confiável;

Utilizar linguagem simples e objetiva.

# Prompt Final

Atue como analista de dados e experiência do cliente no setor bancário.

Sua tarefa é analisar feedbacks de clientes sobre aplicativos bancários, Pix, cartão de crédito e atendimento digital para identificar temas recorrentes, sentimentos dos clientes e oportunidades de melhoria.

# Contexto

A análise será utilizada pela equipe de experiência do cliente para priorizar melhorias nos canais digitais e reduzir problemas relatados pelos usuários.

Dados disponíveis

Serão fornecidos:

Data do feedback;

Canal de atendimento;

Texto do comentário;

Produto citado;

Nota de satisfação de 1 a 5.

# Instruções de Análise

Classifique os feedbacks por tema, sentimento, urgência e produto citado;

Identifique os principais problemas, elogios e oportunidades;

Apresente evidências encontradas nos comentários;

Sugira ações práticas para as equipes responsáveis;

Destaque os impactos observados na experiência do cliente.

# Formato da Resposta

Resumo executivo com até 5 linhas;

Tabela contendo tema, sentimento, evidência e ação recomendada;

Lista com as 3 principais prioridades identificadas;

Recomendações finais.

# Restrições

Utilize apenas os dados fornecidos;

Não invente números ou conclusões;

Não exponha informações pessoais ou sensíveis;

Informe limitações quando houver poucos dados;

Utilize linguagem clara e objetiva para tomada de decisão.

### Conclusão

Com este desafio, pude entender melhor como estruturar prompts mais completos e eficientes. A atividade demonstrou que fornecer contexto, critérios de análise e restrições adequadas ajuda a IA a gerar respostas mais úteis, organizadas e alinhadas ao objetivo desejado.
