# Desafio-criativo-Extraindo-Insights-do-Feedback-de-Clientes-Bancario.
Neste desafio, o objetivo foi criar um prompt capaz de ajudar uma Inteligência Artificial a analisar feedbacks de clientes bancários e transformar essas informações em insights úteis para a tomada de decisão.  A atividade mostrou como a qualidade de um prompt depende de uma boa definição de objetivo, contexto e restrições.
